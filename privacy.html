iiPython Privacy Policy

Last updated on December 10th, 2023.

Preface

Hai! I'm iiPython and am not a lawyer, so take everything listed here with a grain of salt. The following document describes all the data I collect to keep my services running.

What's collected

Manual data collection:

  • Codelet (codelet.codes, inv.codelet.codes)
    • Please see the Codelet Privacy Policy.
    • Cached data is purged every 10 minutes, and all disk data is erased at the start of every month.
  • iiPython Website 2021-2022 (old.iipython.dev)
    • Built-in URL shortener stores the shortcode and destination.
    • This database does NOT have an erase schedule.
  • Public Image Proxy (images.iipython.dev)
    • Image dimensions, color type, and metadata are all cached in RAM.
    • This cache autoclears every hour and no data about images is written to disk.

Automatic collection (all sites):

  • Page visited, request type, http version, and user agent
    • Provides basic analytics about where requests are coming from and going to.
  • IP address
    • Used to block and report IPs that are attempting to harm my services.

Where it's stored

All analytical data is stored on a 100% locally owned server, under my own roof. No third party has access to this data, as I don't share or sell any visitor information.

Additionally, access logs (which contain most of the recorded data) are encrypted and purged once a month.

Data Sharing Exceptions

If there is a lot of malicious traffic coming from a single IP address, I will file an abuse report with the ISP in question (or whoever administered the IP). No data will be shared with them other then the IP address.

Pesky Government

If for some reason the FBI/NSA/Police (or any other US agency) wants access to my service data, they will be immediately declined unless they have a search warrant (since then I can't do anything :c).

However, even if they seize my drives, all user data is encrypted.

CCPA & GDPR Protection Laws

My services are exempt from CCPA and GDPR data protection laws.
However, you can still email me to have your data purged from my drives (even if you don't live in a protected area).

Cloudflare Notice

It's worth noting that I use cloudflare for DNS and DDoS protection. Therefore, the Cloudflare Privacy Policy also applies.
