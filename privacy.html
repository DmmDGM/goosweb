iiPython Privacy Policy

Last updated on November 11th, 2023.

Preface

Hai! I'm iiPython and am not a lawyer, so take everything listed here with a grain of salt. The following document describes all the data I collect to keep my services running.

What's collected

Manual data collection:

  • ROBLOX Gamepass Information (inventory.pixellicio.us)
    • Stored in RAM to speed up subsequent requests
  • URL Destination (old.iipython.dev)
    • Required for the URL shortener to function
  • Image dimensions, color type, and metadata (images.iipython.dev)
    • Image info is stored in RAM to assist with caching
    • No data about images is written to disk

Automatic collection (all sites):

  • Page visited
    • Gives me basic analytics about where requests are going
  • IP address
  • Estimated city, state, and country
    • Used to block and report IPs that are attempting to harm my services

Where it's stored

All analytical data is stored on a 100% locally owned server, under my own roof. No third party has access to this data, as I don't share or sell any visitor information.

Additionally, access logs (which contain most of the recorded data) are encrypted and purged once a month.

There is one exception to these rules however: If there is a lot of malicious traffic coming from an IP, I will file an abuse report with whoever administered the IP. Nothing else is shared besides the ip address though.

Pesky Government

If for some reason the FBI/NSA/Police (or any other US agency) wants access to my service data, they will be immediately declined unless they have a search warrant (since then I can't do anything :c).

However, even if they seize my drives, all user data is encrypted.

CCPA & GDPR

My services are exempt from CCPA and GDPR data protection laws.
However, you can still email me to have your IP address purged from my logs (even if you don't live in a protected area).

Cloudflare

It's worth noting that I use cloudflare for DNS and DDoS protection. Therefore, the Cloudflare Privacy Policy also applies.
